# Tic Tac Toe

## Description
The Tic Tac Toe game is a game for two players, called "X" and "O", who take turns marking the spaces in a 3Ã—3 grid. The player who succeeded in placing three respective marks in a horizontal, vertical, or diagonal row wins the game.
